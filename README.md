# Proyecto: Jmail Clone (Demo)

Este proyecto es una recreación tipo demo de una interfaz de correo electrónico estilo “Jmail”, creada únicamente con fines educativos y de aprendizaje.

## Aviso importante

Este proyecto:
- No está afiliado, asociado ni respaldado por ningún servicio real de correo electrónico.
- Es únicamente una demostración de interfaz y diseño.
- No debe ser utilizado para suplantación de identidad, envío de spam o cualquier actividad ilegal.
- El autor no se hace responsable del uso indebido que terceros puedan darle al proyecto.

## Objetivo

El objetivo de este proyecto es practicar:
- Diseño web (HTML, CSS, JavaScript)
- Maquetación de interfaces tipo email
- Experiencia de usuario (UI/UX)

## Uso permitido

Solo se permite su uso para:
- Aprendizaje personal
- Proyectos educativos
- Demostraciones técnicas

## Uso prohibido

Queda estrictamente prohibido:
- Suplantar servicios reales
- Usar el proyecto para engañar a usuarios
- Actividades ilegales o no éticas
